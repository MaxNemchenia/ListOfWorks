Технология WPF

В приложении используется локальная база данных MSSQL. 
Для подключения  требуется правильно прописать connectionString(если не запустится сразу)
 в конструкторе класса dbConnection, а именно:
-В Data Source указать установленный localdb сервер
-В AttachDbFilename указать путь к базе(в проекте  база расположена в папке .\bindebug,  нызвается Work.mdf)
